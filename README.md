# Hjelm Send

Självhostad fildelning på [transfer.hjelms.com](https://transfer.hjelms.com). Dela filer via länk med valfritt utgångsdatum, lösenordsskydd och besöksstatistik.

## Funktioner

- Dela filer via länk
- Obegränsad filstorlek (begränsas av diskutrymme)
- Utgångsdatum på delningar
- Lösenordsskydd och besöksgräns
- Tidsstämplar per nedladdning
- E-post till mottagare
